## 이동 방향 수정 (Y축 반전)

### 수정 사항
노드의 로컬 좌표계 특성상 `+Y`가 아래쪽을 향하고 있는 것을 확인하였습니다. 따라서 위로 들어 올리는 동작을 구현하기 위해 이동 방향을 반전시켰습니다.

1.  **리프트 방향 변경**: `y: "+=${liftDist}"` → `y: "-=${liftDist}"`
    *   이제 댐퍼 홀더가 의도한 대로 위쪽으로 들리게 됩니다.

### 수정된 코드 (`DamperCoverAssemblyService.ts`)

```typescript
// 1단계: Y축 방향 리프트 (방향 반전 적용)
tl.to(assemblyNode.position, {
    y: `-=${liftDist}`, // -= 로 변경하여 위로 이동
    duration: liftDur,
    ease: 'power2.out'
});
```

이제 다시 실행하여 홀더가 올바른 방향(위쪽)으로 들리는지 확인해 주시기 바랍니다. 만약 슬라이드 방향(Z축)도 반대라면 마찬가지로 기호를 변경하여 조정할 수 있습니다.
