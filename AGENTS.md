## Rules
- All code prioritizes readability, and functions adhere to the single responsibility principle.
- When adding new features, always maintain the style of the existing code.
- Commit messages follow the Conventional Commits format.
- Please answer my question in Korean.
- You don't need to share the entire modified code; just the modified portion is sufficient.
- I'll write the question in the 000.md file. Please write your answer in the answer.md file. 
- When writing your answer, please overwrite the existing content in the answer.md file.
- There's no need to print your answer in the command prompt window.
- When providing answers, please write in Markdown style, but in a structured way that is easy to read at a glance.
