시각화 선이 위로 향하는 문제를 해결하기 위해 다음과 같이 수정하였습니다.

### 1. 수정 내용
- **Y축 오프셋 부호 변경**: 현재 프로젝트 환경에서는 Y축 양의 방향(`+Y`)이 시각적인 "아래"에 해당함을 확인하였습니다. 이에 따라 오프셋 값을 `-0.05`에서 `+0.05`로 변경하였습니다.

### 2. 수정된 코드 로직
```typescript
// 머리 중심(headCenter)에서 Y축 양의 방향으로 0.05m 지점 계산
const downwardPoint = headCenter.clone().add(new THREE.Vector3(0, 0.05, 0));

// 시각화 실행
visualizer.visualizeAssemblyPath(headCenter, downwardPoint);
```

### 3. 결과 기대
- 이제 스크류 머리 중심점에서부터 실제 화면상의 아래 방향으로 선이 그려지게 됩니다.
- `animateScrewLinearMove` 및 `animateScrewLinearMoveToDamperCaseBody` 두 함수 모두에 적용되었습니다.
